# Animal SOS

## 📌 Sobre o projeto
O **Animal SOS** é um sistema web desenvolvido como **Trabalho de Conclusão de Curso (TCC)**, com o objetivo de auxiliar no registro, acompanhamento e visibilidade de denúncias de animais em situação de abandono, maus-tratos ou risco.
A proposta do sistema é conectar a comunidade, ONGs e responsáveis, facilitando o processo de denúncia e acompanhamento dos casos.

## 🎯 Objetivo do projeto
Desenvolver uma aplicação que permita:
- Registrar denúncias de animais em situação de risco
- Acompanhar o status das denúncias
- Facilitar a comunicação entre usuários e responsáveis

## 🛠️ Tecnologias utilizadas
- PHP (arquitetura MVC)
- HTML5
- CSS3
- JavaScript
- Banco de dados relacional (MySQL)
- Servidor local (XAMPP)

## 📐 Funcionalidades previstas
- Cadastro e login de usuários
- Registro de denúncias com imagem e descrição
- Listagem de denúncias
- Visualização detalhada da denúncia
- Comentários nas denúncias
- Controle de status (Aguardando, Adoção)
- Localização de denúncias através de filtros
- Perfil de usuário
